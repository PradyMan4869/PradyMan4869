## 🚀 Some of my Projects!

**[SC-GraphRAG-Engine](https://github.com/PradyMan4869/SC-GraphRAG-Engine)**
> Fully local GraphRAG pipeline over 782 Indian Supreme Court judgments — graph-augmented retrieval with multi-hop reasoning, four query modes, and a PyTorch-CUDA inference stack. No external API calls.
> `Python` `LightRAG` `Neo4j` `PyTorch` `RAG`

---

**[A2A Portfolio Construction](https://github.com/PradyMan4869/A2A_Portfolio-Construction)**
> 6 AI agents communicating with each other using Google's Agent2Agent (A2A) protocol for Indian equity portfolio construction — XGBoost signal generation (78% walk-forward accuracy), LSTM price forecasting, FAISS semantic stock search, and vectorbt backtesting across 6,999 stocks.
> `Python` `XGBoost` `LSTM` `FAISS` `A2A Protocol` `ML`

---

**[Chaturanga Bot](https://github.com/PradyMan4869/Chaturanga_Bot)**
> AI bot for Chaturanga, the 600 CE ancestor of chess — implements authentic rules, 15 difficulty levels from random play up to NNUE depth-12 search, and a hand-curated puzzle set.
> `Python` `PyTorch` `NNUE` `Alpha-Beta Search`

---

**[CNN-LSTM Anomaly Detection](https://github.com/PradyMan4869/Video-Frame-Analysis-and-Anomaly-Detection-with-CNN-LSTM)**
